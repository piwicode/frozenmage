Frozen Mage

A small platformer made in 5 weeks by two random guys.

Download .zip View on GitHub

Classical platformer

Frozen Mage is a classic platformer telling the story of an unfortunate magician trying to pass his final exam.

5 weeks for a game

This game was made as in 5 weeks by a team of two. We consider it more as a draft. We just wanted to try our hands at GameMaker and get some experience. Part of the challenge was to craft all the assets: graphics, sounds and code (apart from Game Maker engine).

Controls

With a keyboard, use left and right arrows to run in that direction, space to jump, and the shift key to fire once unlocked. The up and down arrows control vertical scrolling, and fire direction. Use F11 or the F key to toggle fullscreen.

If for some reason to get stuck, it is possible to restart the current level by pressing the R key.

With a gamepad, use the thumb-stick or the d-pad in place of the keyboard arrows, the button A to jump, and the button X to fire.

Download

There is a Window binary available, and the HTML5 version is on its way. If your platform is not available, consider building it from sources.

Tools of the trade

We use Game Maker to build the game. Graphics are made with Blender and Photoshop. For sound we use Reaper, and we use Google Analytics to know if people play this game.
